name: check-rules
description: Check whether a proposed change violates the project rules
---

I will describe a proposed change. Your job:

1. Read `.agents/rules/01-project-rules.md` and `CLAUDE.md` if not already in context
2. Identify any rule the proposed change conflicts with
3. State the conflict explicitly — quote the rule
4. Propose either:
   (a) a way to do the change that respects the rule, or
   (b) the ADR that would need to exist to override the rule

Do not write any code in this skill. The output is analysis only.
