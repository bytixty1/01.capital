Write a new ADR using the template at `docs/decisions/TEMPLATE.md`.

Procedure:
1. Read the template
2. Read the most recent ADR to get the next number (e.g., if 0003 exists, this is 0004)
3. Ask me for: the decision title, the context, and the proposed direction
4. Draft the ADR with Status: "Proposed"
5. Save as `docs/decisions/NNNN-kebab-case-title.md`
6. Show me the draft for review before committing

Constraints:
- ADRs are short (one page maximum)
- Cite specific articles of the Saudi Companies Law if the decision is law-touching
- Reference any project rules being affected
- Do not mark Status: "Accepted" unless I explicitly tell you to
